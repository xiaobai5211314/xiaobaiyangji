# 项目领域上下文

本文件为需要理解"小白养基"业务语义的 Agent 技能提供领域知识。

## 项目简介

"小白养基" 是一个基金和股票投资管理应用：
- **基金持仓管理**：通过蚂蚁基金 App 截图 OCR 导入持仓，跟踪收益
- **股票持仓管理**：OCR 导入或手动录入股票持仓
- **收益统计**：每日归档、盈亏日历、首页 summary 仪表盘
- **盘中估值**：实时抓取东方财富基金估值，提供临时参考

技术栈：
- 后端：ASP.NET Core 8.0 (C#)，MySQL + Redis
- 小程序前端：uni-app / Vue 3 / TypeScript
- 静态 WebApp：`wwwroot/index.html`（单文件）+ `wwwroot/v2/`（Vite 构建）

## 核心领域实体

| 实体 | 说明 |
|------|------|
| `MyFundConfig` | 用户基金配置，含蚂蚁确认金额、昨日收益、持有收益等字段 |
| `FundData` | 基金净值历史（东方财富抓取） |
| `DailyArchive` | 每日收益归档，正式历史档案 |
| `FundValuationEstimate` | 盘中估值，临时数据，不得进入正式日历 |
| `FundTradeOrder` | 基金交易订单 |
| `StockHolding` | 股票持仓 |
| `StockWatchItem` | 股票自选 |
| `User` | 用户，认证通过 `username` 参数（无 JWT） |

## 基金金额口径强约束

以下规则是硬约束，任何 Agent 修改涉及金额/收益的代码时必须遵守：

1. **蚂蚁基金 App 当前金额** 是平台当前持仓金额的最高优先来源，不等于基金公司/销售机构已经确认的正式持仓金额。
2. 当前金额与确认持仓金额必须拆开：`平台当前金额 = 已确认持仓金额 + 买入待确认金额`。
3. **蚂蚁 OCR 快照是截图展示事实源**。当日有蚂蚁 OCR 快照时，首页当前展示以 OCR 快照为准；正式历史档案只能写入已确认持仓金额和已确认收益。
4. **昨日收益** 只能用于以下两个场景：
   - "昨日确认收益" 展示
   - "盈亏日历" 当日收益记录
5. **禁止** 用 `当前金额 - 昨日收益` 反推蚂蚁确认金额。
6. **禁止** 用 `持有收益 - 昨日收益` 反推蚂蚁持有收益。
7. **盘中估值** 只能作为临时估算，不能写入正式历史日历。

依据：
- 中国证监会深圳监管局投资者教育材料：15:00 前后的交易申请归属不同交易日，交易确认日通常为 T+1，部分特殊基金可能为 T+2/T+3。
- 天天基金帮助中心：T 日以 15:00 为界，15:00 前提交按当天收市后净值成交，通常 T+1 确认。
- 富途基金持仓资产说明：基金申购资金在交易确认前未形成持仓份额，会以买入待确认金额形式增加基金持仓金额。

## 盈亏日历口径

- `DailyArchive` 是历史收益的正式档案表。
- 正式历史档案优先来自蚂蚁 OCR 确认数据。
- 缺少蚂蚁确认时，官方净值可以作为 `official-nav-pending` 待确认档案。
- 盘中估值 **不得** 进入正式日历。
- `DailyArchive` 中 `FundCode = 'TOTAL'` 的汇总行是首页"昨日确认收益"的优先读取来源。

## 首页 summary 口径

首页仪表盘展示以下关键数字，计算公式如下：

| 字段 | 计算方式 |
|------|---------|
| 账户总金额 | 蚂蚁当前金额合计（包含买入待确认金额） |
| 确认持仓金额 | 蚂蚁当前金额合计 - 买入待确认金额 |
| 买入待确认金额 | 蚂蚁 App 显示或用户确认的买入待确认金额 |
| 昨日确认收益 | 最新 DailyArchive TOTAL 行的 DailyProfit |
| 昨日确认日期 | 最新 DailyArchive TOTAL 行的 RecordDate |
| 蚂蚁持有收益 | 蚂蚁 App 当前持有收益合计 |
| 今日盘中估算 | 盘中估算市值 - 确认持仓金额 |
| 今日估算收益率 | 今日盘中估算 / 确认持仓金额 × 100 |

## 收益率计算口径

```
持仓本金 = 确认持仓金额 - 持有收益
持有收益率 = 持有收益 / 持仓本金 × 100%
今日收益率 = 今日收益 / 昨日基准金额 × 100%
```

**禁止**为了让页面数字看起来接近而修改上述公式。如果数字对不上，先排查数据来源，不要改公式。

## 领域术语

| 术语 | 含义 |
|------|------|
| 蚂蚁确认数据 | 蚂蚁基金 App 截图经 OCR 提取后确认的数据 |
| 蚂蚁当前金额 | 蚂蚁 App 显示的当前持有金额 |
| 蚂蚁持有收益 | 蚂蚁 App 显示的累计持有收益 |
| 昨日收益 | 蚂蚁 App 显示的前一日确认收益 |
| 盘中估值 | 东方财富 API 返回的实时基金估值（非确认数据） |
| 官方净值 | 基金公司公布的正式净值（T+1 确认） |
| DailyArchive | 每日收益归档表，盈亏日历的数据源 |
| TOTAL 汇总行 | DailyArchive 中 FundCode='TOTAL' 的用户级汇总行 |
